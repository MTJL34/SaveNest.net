categories
Create : POST http://localhost:3000/api/categories
Read : GET http://localhost:3000/api/categories
Read by ID : GET http://localhost:3000/api/categories/ID
Unlock private category : POST http://localhost:3000/api/categories/ID/unlock
Update : PATCH http://localhost:3000/api/categories/ID
Delete : DELETE http://localhost:3000/api/categories/ID
